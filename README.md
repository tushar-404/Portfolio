# Tushar Kumar Singh — Portfolio

Live Site: https://portfolio-ten-hazel-82.vercel.app/

A modern, single-page developer portfolio built with semantic HTML, custom CSS, animations, and interactive JavaScript. The site showcases an about section, technology stack, professional links, and a dedicated projects page.

## Features
- Smooth, elegant UI with Catppuccin-inspired dark theme
- Animated hero with typing effect and interactive 3D hover cards
- Sticky education/summary block with subtle transitions
- Tech docs-style sections listing tools and stacks
- Quick access links to GitHub, LinkedIn, LeetCode, Codeforces, and Projects page
- Projects page with animated cards, filters, and tilt effects
- Fully responsive (mobile-first), accessible markup

## Pages
- Home: index.html — main profile, about, links, and tech documentation
- Projects: projects.html — curated live project links and descriptions

## Quick Start
- Open index.html directly in your browser
- Or host via any static server (e.g., Vercel, Netlify, GitHub Pages)

## Customize
- Profile image: replace photo_2025-11-02_21-48-06.jpg in the root folder
- Name and meta: update <title> and hero text in index.html
- Links: update anchor hrefs in the Links section of index.html
- Projects: edit the projectsData array in projects.html

## Tech Stack
- HTML5
- CSS3 (custom, no framework; utility-inspired classes)
- JavaScript (vanilla)
